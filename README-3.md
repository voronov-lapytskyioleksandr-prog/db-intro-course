**Лабораторна робота 3: Маніпулювання даними SQL (OLTP)**

*1. Опис виконаних запитів*
У цій лабораторній роботі було протестовано базу даних системи замовлень ресторану шляхом виконання транзакційних SQL-запитів (OLTP). Усі запити виконані успішно.

* **SELECT (Отримання даних):** Було виконано вибірку всіх страв, ціна яких перевищує 100 грн, з вказанням лише конкретних стовпців (назва та ціна). Також було знайдено всі замовлення для конкретного столика.
* 
**INSERT (Додавання даних):** Оператор INSERT використовується для додавання нових рядків до таблиці. Було додано нову категорію "Десерти" та нову страву "Торт Наполеон", яка прив'язана до цієї категорії.


* 
**UPDATE (Зміна даних):** Оператор UPDATE змінює дані. За допомогою SET та WHERE було безпечно змінено ціну новоствореного десерту з 120.00 на 135.00.


* 
**DELETE (Видалення даних):** Оператор DELETE видаляє рядки з таблиці, і без WHERE всі рядки будуть видалені. Було безпечно видалено тестову страву "Торт Наполеон" за її назвою.



*2. Тестування та результати*
Усі SQL-запити були успішно виконані у клієнті pgAdmin. Після операцій INSERT, UPDATE та DELETE виконувалися перевірочні запити SELECT, щоб переконатися, що зміни в базі даних відбулися коректно і вплинули лише на потрібні рядки. Докази виконання (скріншоти з результатами запитів) додані нижче.

*3. SQL-скрипт з DML-запитами*

```sql
-- 1. SELECT: Отримання даних
-- Вибірка назв та цін страв, які дорожчі за 100 грн
SELECT name, price 
FROM dish 
WHERE price > 100;

-- Вибірка всіх замовлень для столика номер 3
SELECT order_id, total_amount, order_date 
FROM restaurant_order 
WHERE table_number = 3;


-- 2. INSERT: Додавання нових даних
-- Додаємо нову категорію
INSERT INTO category (name, description) 
VALUES ('Десерти', 'Солодкі страви та випічка');

-- Додаємо нову страву в цю категорію (припускаємо, що ID нової категорії = 4)
INSERT INTO dish (name, description, price, weight, category_id) 
VALUES ('Торт Наполеон', 'Класичний торт з кремом', 120.00, 150, 4);

-- Перевіряємо, чи додалася страва
SELECT dish_id, name, price 
FROM dish 
WHERE name = 'Торт Наполеон';


-- 3. UPDATE: Оновлення існуючих даних
-- Змінюємо ціну десерту
UPDATE dish 
SET price = 135.00 
WHERE name = 'Торт Наполеон';

-- Перевіряємо, чи змінилася ціна
SELECT name, price 
FROM dish 
WHERE name = 'Торт Наполеон';


-- 4. DELETE: Видалення даних
-- Безпечно видаляємо створений десерт
DELETE FROM dish 
WHERE name = 'Торт Наполеон';

-- Перевіряємо, що рядок зник (запит має повернути порожній результат)
SELECT * FROM dish 
WHERE name = 'Торт Наполеон';

```
